Tool coverage

Inspect, diagnose, and manage Windows from your MCP client.

Task Manager

CPU, GPU, memory, disk usage, process trees, process details, suspend, resume, and kill.

Storage Cleanup

Disk reports, largest files, duplicate discovery, old files, temp cleanup, and Recycle Bin workflows.

Security

Defender status and scans, firewall rules, BitLocker, Secure Boot, UAC, updates, and restore points.

Hardware

GPU details, display configuration, installed drivers, problem devices, battery, thermal, and power plans.

Networking

Interfaces, active sockets, listening ports, firewall correlation, Wi-Fi networks, and latency checks.

Developer Workflows

Python and Node discovery, environment variables, common development ports, screenshots, and clipboard helpers.

Safety model

Built for local control, with powerful tools clearly separated by risk.

The server runs with the privileges of the account that starts it. Read-only reports should be used first, dry-run workflows are preferred before mutation, and sharp actions deserve explicit review.

  • Prefer report tools before changing system state.
  • Use Recycle Bin and dry-run cleanup tools before permanent deletion.
  • Avoid Administrator launch unless the task needs elevated privileges.
  • Do not expose the server over a network without authentication and a tool allowlist.
